*Пошаговая Игра “РОГАЛИК” на JavaScript*

### Как запустить?
Просто загрузить всю папку `rogue` и открыть файл `index.html` в браузере
### Управление
Перемещение:
- `W` - Наверх
- `S` - Вниз
- `A` - Влево
- `D` - Направо

Действия:
- `Space` - Нанести удар

### Предметы
- `Красная колба` - восполнить здоровье до 100
- `Меч` - увеличить урон в 2 раза
